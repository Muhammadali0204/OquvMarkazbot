# OquvMarkazbot
Kerakli kutubxonalarni o'rnatish
```shell
$ pip install -r requirements.txt
```

`.env` fayl yaratish
```shell
$ cp .env.dist .env
```

`.env` fayliga kirib bot tokeni va boshqa parametrlarni sozlash
```shell
$ nano .env
```


## Dasturni ishga tushirish
```shell
$ python app.py
```
